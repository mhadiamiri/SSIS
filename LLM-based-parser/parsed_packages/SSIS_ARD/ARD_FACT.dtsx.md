## 3. Package Flow Analysis

*   The package starts with an Expression Task that seems to always evaluate to true.
*   It then executes two packages in sequence: `ARD_PRIME_FACT.dtsx` and `ARD_OPERA_FACT.dtsx`.
*   The `OnError`, `OnPostExecute`, and `OnPreExecute` event handlers are configured to update the `ETL_RUN_STATUS` table.
